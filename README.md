This mod adds **6 new weapons** and **2 charms** with useful power to Minecraft:

**Katana** - 6 Attack Damage; 1.4 Attack Speed

**Dagger** - 3 Attack Damage; 2.2 Attack Speed

**Rapier** - 4 Attack Damage; 1.7 Attack Speed

**Battle Axe** - 10 Attack Damage; 1.1 Attack Speed

**Mace** - 8 Attack Damage; 0.9 Attack Speed

**Giant Sword** - 15 Attack Damage; 0.3 Attack Speed

**Fireball Charm** - Shoots Fireballs

**Rain Charm** - Causes rain for 5 XP Levels
